# Vanilla App Template

Цей проект було створено за допомогою Vite. Для знайомства та налаштування
додаткових можливостей [звернись до документації](https://vitejs.dev/).

EduConf — 2025
Командний проєкт: Платформа для пошуку та реєстрації на освітні заходи.

📝 Про проєкт
Це веб-додаток, який допомагає користувачам дізнаватися про навчальні конференції, знайомитися зі спікерами та реєструватися на події. Проєкт розроблений у складі команди з використанням сучасних методологій верстки.

🛠 Моя роль та внесок
Я відповідала за розробку Success Modal — критичного елемента інтерфейсу, який підтверджує успішну реєстрацію користувача.

Мої основні технічні задачі:
Семантична верстка: Створення структури модального вікна з використанням HTML5 та SVG-спрайтів для іконок.

CSS-архітектура: * Використання CSS-змінних (:root) для створення гнучкої системи кольорів та швидкого рестайлінгу.

Реалізація темного Backdrop (оверлею) з ефектами плавного зникнення/появи через transition.

Адаптивність (Mobile First):

Верстка була реалізована для трьох типів пристроїв: Mobile , Tablet(768px+)  та Desktop (1440px+).

Використання тегу <picture> для адаптивної графіки: автоматична підміна зображень залежно від ширини екрана та підтримка дисплеїв високої чіткості (Retina @2x).

UI/UX деталі: * Налаштування інтерактивних станів для кнопок (hover, focus, active).

Забезпечення правильного позиціювання модального вікна (fixed, inset: 0) та керування скролом.

📁 Використані технології
HTML5 — семантика та медіа-запити.

CSS3 (SCSS) — змінні, вкладеність, адаптивність.

SVG Sprites — для іконки закриття вікна.
